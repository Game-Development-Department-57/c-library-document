割り当て規則

err型(int型)の32bitを次のように利用する。 範囲: 幅: 用途 31bit: 1bit: 正負のフラグ 30-24bit: 7bit: 予約 23-12bit:12bit: クラス 11- 0bit:12bit: 詳細

基本的にエラーの割り当ては、負の値であるため正の値が割り当てられるのは特殊な場合である。 0は必ずSUCCESSである。

ID NAME 意味

0x00000000 SUCCESS 正常
0x80000000 ERROR 異常

0x80001001 ALLOCNULL ALLOCできない
